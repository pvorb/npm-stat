npm-stat

npm-stat can generate download charts for any package on npm.

Package:

These statistics are not provided in real-time. All numbers will change at most once per day.

If there are other issues with the service, please create an issue at Github.

To keep this site running and ad-free, I would appreciate a donation.

All data comes directly from npm.

The charts of this service are powered by Highcharts JS which is provided under a CC BY-NC 3.0 licence.
